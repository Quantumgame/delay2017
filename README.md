### Requirements

 - Python 2.7.3
 - nengolib==0.4.1
 - nengo==2.4.0
 - scipy==0.18.0rc1
 - numpy==1.11.3
 - seaborn==0.7.1
 - matplotlib==2.0.0
 - doit>=0.29.0

### Instructions

 - Install all requirements.
 - Run `doit` to generate all of the files required by LaTeX.
 - Then compile the LaTeX paper!

### Source Code

All of the optimization/simulation/analysis code is in `main.py` (and executed by `dodo.py`).
